## Descripción
En esta práctica se ha configurado un sistema de **Validación Automática** utilizando GitHub Actions. 
Cada vez que se sube código al repositorio, el sistema revisa la sintaxis de Terraform.

## Estado del Proyecto
El badge superior indica si la última validación fue exitosa (**passing**) o si contiene errores (**failing**).
